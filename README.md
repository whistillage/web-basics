## 1. HTML 기본 태그
- 기본 구조
    <!DOCTYPE html>
    <html>
        <head>
            <title></title> : 페이지 제목
            <meta charset="utf-8">
        </head>
        <body>
        </body>
    </html>

- <strong></strong> : 굵게
- <h1> ~ <h6></h1> ~ </h6> : 제목 강조
    1이 가장 크고 6이 가장 작음
- <u></u> : 밑줄
- <p></p> : 단락
    css를 이용한 꾸미기 용이
- <img> : 이미지
    src = "directory" : 이미지의 출처
    width = "100%" : 이미지의 크기
- <ol></ol>, <ul></ul> : 정렬, 비정렬 목록
    <li></li> : 자식 태그, 내용
- <table> : 표
    <tr></tr> : 자식 태그, 행
        <td></td> : 자식 태그, 열
